# Contributing to Strata Unreal Engine Plugin

## Setup

1. Navigate to your Unreal Engine project's plugins folder (Ex: `cd "C:\Documents\Unreal Projects\Project1\Plugins"`)
2. Run `git clone https://github.com/UltraDagon/StrataPluginUE`
3. Enter the directory: `cd StrataPluginUE`

## Creating Pull Requests

1. Before beginning your feature, run `git checkout main; git pull` to ensure your version is up to date.
2. Create a new branch from main: `git checkout -b <branch-name>`. Branch name formatting: `[feature, fix, test]/title`
3. Add your changes: `git add .`
4. Commit your changes: `git commit -m "Descriptive description"`
5. Push your changes: `git push -u origin <branch-name>`
6. Visit [https://github.com/UltraDagon/StrataPluginUE](https://github.com/UltraDagon/StrataPluginUE) in your browser and click "Create Pull Request".
